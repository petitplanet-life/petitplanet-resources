# Petit Planet Resources

Player-friendly Petit Planet guides, wiki pages, database tools, and daily checklists from [PetitPlanet.life](https://petitplanet.life/).

PetitPlanet.life focuses on practical pages for players: release status, download notes, platforms, codes, neighbor gifts, fish, furniture, cooking recipes, events, patch notes, and daily reset checks. This repository is a public index for the main Petit Planet guide pages that are useful to open again while playing.

## Start Here

- [Petit Planet Guide](https://petitplanet.life/) - main site
- [Petit Planet Wiki](https://petitplanet.life/wiki/) - guide and database index
- [Beginner Guide](https://petitplanet.life/beginner-guide/) - first steps for new players
- [Patch Notes](https://petitplanet.life/patch-notes/) - official update and test-note tracker
