# Petit Planet Guide Resources

[PetitPlanet.life](https://petitplanet.life/) is a dedicated Petit Planet guide, wiki, database, and daily tracker for players who want practical game information in one place.

The site is built around real player questions: is Petit Planet playable yet, where should players download it, which platforms have test references, what changed in official test notices, and which tools help with daily planning. Launch and access information is kept separate from reference databases, so players can review official status updates without mixing them with test-build item data.

## Main Site

- [Petit Planet Guide](https://petitplanet.life/) - the main hub for release status, tools, databases, and daily checks
- [Petit Planet Wiki](https://petitplanet.life/wiki/) - a route index for every major guide and database page
- [Beginner Guide](https://petitplanet.life/beginner-guide/) - a first-route guide for new players checking access, systems, and repeat tasks
- [Patch Notes](https://petitplanet.life/patch-notes/) - official reveal, Coziness Test, and Stardrift Test update notes
